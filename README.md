merge_split.py 
==============
youtube動画：https://www.youtube.com/watch?v=DZLrn0pP4hU<br />
実行方法：`python merge_split.py` <br />
利用したライブラリ：PyPDF2<br />
インストール方法：`pip install pypdf2`<br /><br />

split()関数<br />
第一引数：分割したpdf名<br />
第二引数：分割したpdfを格納するディレクトリ<br />
返り値；なし<br />
<br />
merge()関数<br />
第一引数：結合したいpdfが格納されたディレクトリ<br />
第二引数：結合pdfを格納するディレクトリ<br />
返り値：なし<br />
